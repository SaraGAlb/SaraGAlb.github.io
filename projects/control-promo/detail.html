Proyecto

Control Promo Mensual

HTML CSS JavaScript

Descripción

Herramienta web para gestionar el reparto diario de propinas entre empleados. Permite registrar la cantidad total del día, seleccionar qué empleados trabajaron y cuántas horas, calculando automáticamente la parte proporcional que corresponde a cada uno. Los datos persisten entre sesiones mediante localStorage y pueden exportarse en PDF o Excel al cierre del ciclo mensual (del 21 al 20).

Retos y aprendizajes

El principal reto fue diseñar una lógica de reparto proporcional por horas que fuera robusta ante distintas combinaciones de empleados y jornadas. También implementé validaciones para evitar duplicar registros de un mismo día, con la opción de cargar y editar datos ya guardados. Para las exportaciones integré las librerías jsPDF y SheetJS, generando documentos con formato profesional listos para entregar.
