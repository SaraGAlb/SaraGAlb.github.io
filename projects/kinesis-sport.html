Kinesis Sport

Sitio web para un centro de entrenamiento personal — 1er proyecto

HTML5 CSS3 Responsive 1er proyecto Sin frameworks

Descripción

Sitio web completo para un centro de entrenamiento personal ficticio llamado Kinesis Sport. Desarrollado durante el primer curso como ejercicio práctico de maquetación web, sin usar frameworks ni librerías de CSS.

Páginas y funcionalidades

  • Página de inicio con hero, servicios y formulario de contacto
  • Página de entrenamiento con tarjetas y distintas modalidades
  • Página de recuperación con contenido detallado
  • Página "Sobre mí" con biografía del entrenador
  • Página de contacto con mapa y datos del local
  • Menú hamburguesa responsive sin JavaScript (solo CSS)
  • Footer completo con enlaces e iconos de redes sociales

Lo que aprendí

  • Estructura semántica HTML5 (header, nav, main, article, footer)
  • Diseño responsive con media queries
  • Flexbox para layouts complejos
  • Menú móvil con el truco del checkbox en CSS puro
  • Organización de un proyecto con múltiples páginas
⚠️ El formulario de contacto utiliza PHP y no está activo en esta demo estática. Funciona correctamente en un entorno con servidor.
