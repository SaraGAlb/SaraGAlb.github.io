# Portfolio Personal — Sara García Albandea

Portfolio personal desarrollado como proyecto de 2º de DAM para mostrar habilidades técnicas y proyectos realizados durante la formación.

## Páginas

- **Inicio** (`index.html`) — presentación y proyectos destacados
- **Sobre mí** (`about.html`) — biografía, formación y habilidades técnicas
- **Proyectos** (`projects/index.html`) — listado completo de proyectos
- **Detalle de proyecto** (`projects/proyecto-ejemplo.html`) — descripción detallada de cada trabajo

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 — Flexbox, Grid, variables CSS, animaciones, diseño responsive
- JavaScript vanilla — manipulación del DOM, eventos
- Google Fonts — Playfair Display + Lato

## Estructura del proyecto

```
portfolio/
├── index.html
├── about.html
├── projects/
│   ├── index.html
│   └── proyecto-ejemplo.html
├── css/
│   ├── style.css       # Estilos base, variables y layout global
│   └── components.css  # Componentes reutilizables (cards, botones, skills)
└── js/
    └── main.js         # Menú móvil y scroll suave
```

## Cómo ejecutarlo localmente

No requiere instalación ni dependencias. Basta con clonar el repositorio y abrir `index.html` en el navegador:

```bash
git clone https://github.com/tu-usuario/portfolio.git
cd portfolio
```

Para evitar problemas con rutas relativas se recomienda usar un servidor local, por ejemplo con la extensión **Live Server** de VS Code.
